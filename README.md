# Explicación del proyecto
### Backend:
Se realizó en python (flask)

Tiene:

-Registro de usuarios

-Inicio de sesión con JWT

-Administración de perfil (editar datos como, nombre, apellido, foto de perfil,etc) y la eliminación de este.

Para la instalación se debe ejecutar el comando ```pip3 install -r requirements.txt```

Aunque no es necesario,ya que tanto el backend como la base de datos se encuentran en la nube.

### Frontend:
Se realizó en Angular, consumiendo la API del back que se realizó.

Para su instalación solo es necesario ejecutar el comando ```npm install```

Luego ```ng serve```

¡ y a navegar !
